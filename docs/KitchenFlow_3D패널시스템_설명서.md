# KitchenFlow — 3D 패널 주방 레이아웃 시스템

## 개념

주방을 **수평 패널 여러 장**으로 분해한다. 패널들을 평면으로 펼쳐서 편집하고, 종이접기처럼 접으면 실제 주방의 3D 단면도가 만들어진다.

## 작동 방식

### 편집 모드 (펼침)

패널들이 위에서 아래로 평면 전개된다. 배경에 주방 사진을 깔고, 그 위에 패널들이 겹쳐 표시된다.

```
  ┌─────────────────────────────────────┐
  │         패널 1                       │
  ├─────────────────────────────────────┤
  │         패널 2                       │
  ├─────────────────────────────────────┤
  │         패널 3                       │
  └─────────────────────────────────────┘
```

- 패널 위에 주방 장비를 드래그 앤 드롭으로 배치
- 장비끼리, 장비와 패널 경계 간 자석 정렬 (10px 이내 스냅)
- 장비 클릭 시 리사이즈, 복제, 삭제 가능
- 패널 높이는 드래그로 개별 조절, 패널 수는 추가/삭제 가능

### 미리보기 모드 (접힘)

버튼 하나로 패널들이 접힌다. 상단 패널이 기준면이 되고, 그 아래 패널들이 번갈아 앞으로/아래로 90도씩 접혀서 주방 단면도를 형성한다.

```
         ┌──────────────────┐
         │     패널 1        │  ← 기준면 (살짝 기울어짐)
         └──────────────────┤
         ┌──────────────────┘
        ╱                    ╲
       ╱       패널 2         ╲   ← 앞으로 90° 접힘 (수평)
      ╱______________________╲
      │                      │
      │       패널 3          │  ← 아래로 90° 접힘 (수직)
      └──────────────────────┘
```

- 위에서 비스듬히 내려다보는 시점 (기본 45도, 조절 가능)
- 접기/펼기는 0.6초 CSS 애니메이션으로 전환
- 접힌 상태에서 장비 편집 불가, 구조물 상하 이동만 가능

### 기술 구현

- 순수 CSS 3D (perspective + rotateX). Three.js 등 3D 라이브러리 없음
- 패널들은 HTML div 중첩(부모-자식) 구조. 자식 패널이 부모 패널 하단에서 회전
- 편집과 미리보기가 동일한 DOM 구조를 사용하며, rotateX 값만 전환하여 애니메이션 구현

## 현재 상태

어드민 페이지 "주방 레이아웃" 탭에서 프로토타입 동작 중. DB 저장, 인게임 연동은 미구현.
