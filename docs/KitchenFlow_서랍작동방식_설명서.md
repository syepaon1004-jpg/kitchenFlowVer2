# KitchenFlow — 서랍 작동 방식

## 개념

미리보기(접힘) 모드에서 서랍 장비를 클릭하면 실제 서랍처럼 앞으로 열린다. 서랍 face가 앞으로 나오면서 내부 판이 수직 방향으로 스르륵 슬라이드된다.

## 구조

서랍은 3개 레이어로 구성된다:

```
1. 컨테이너 (preserve-3d 공간)
   ├── 2. 내부 판 (빨간 박스) — 서랍과 수직 방향으로 눕혀진 판
   └── 3. 서랍 본체 (face) — 서랍 정면
```

## 작동 원리

### 닫힌 상태 (단면)

```
  패널 │서랍│     ← 내부 판은 패널 표면(Z=0)에 숨어있음
```

### 열린 상태 (단면)

```
  패널 ═══════│서랍│   ← 서랍 face: translateZ(100px)
       ↑
    내부 판: translateZ(100px) + rotateX(-90deg) 고정
```

### 각 레이어의 transform

| 레이어 | 닫힌 상태 | 열린 상태 |
|--------|----------|----------|
| 서랍 본체 | translateZ(0) | translateZ(100px) |
| 내부 판 | translateZ(0) rotateX(-90deg) | translateZ(100px) rotateX(-90deg) |

- `rotateX(-90deg)`는 **고정값**. 내부 판의 방향(서랍과 수직)을 결정할 뿐, 애니메이션하지 않음
- `translateZ` 값만 변화하여 서랍과 같은 Z방향으로 슬라이드
- transform 순서가 중요: `translateZ` → `rotateX` (이동 먼저, 방향 나중)

## 이벤트 처리

CSS 3D 회전된 요소는 브라우저 hit-test에서 클릭 대상으로 인식되지 않는다. 서랍 클릭은 scene 레벨에서 `getBoundingClientRect()` 좌표 비교로 감지한다.

## 현재 상태

프로토타입 단계. 내부 판은 빨간 네모 박스. 이후 실제 서랍 내부 이미지로 교체 가능.
